# BOL Bracket 2026 🏀

BOL Agency internal March Madness bracket game.

## Stack
- React + Vite (frontend)
- GitHub Pages (hosting)
- Supabase (auth + database)

## Local Dev
```bash
npm install
npm run dev
```

## Deploy
Push to `main` — GitHub Actions auto-deploys to GitHub Pages.

## Live URL
`https://[bol-org].github.io/bol-bracket-2026/`
